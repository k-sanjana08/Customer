## Project Workflow

### 1. Data Loading
- Loaded raw dataset into Python using Pandas  
- Verified schema, data types, and overall structure  

### 2. Exploratory Data Analysis (EDA)
- Analyzed data distributions, trends, and outliers  
- Identified missing values and inconsistencies  
- Generated summary statistics and visualizations  

### 3. Data Cleaning & Preparation
- Handled missing and duplicate records  
- Standardized column names and formats  
- Created derived features to support analysis  

### 4. SQL Analysis
- Imported cleaned data into PostgreSQL
- Wrote SQL queries to answer business questions  
- Used joins, aggregations, filtering, and window functions  

### 5. Dashboard Development
- Built an interactive **Power BI dashboard**  
- Included KPIs, trends, and drill-down views  
- Designed with business users in mind  

### 6. Reporting & Presentation
- Created a structured analytical report summarizing insights  
- Focused on clarity, storytelling, and actionable outcomes
